# Flow 💪

A mobile-first PWA workout tracker to log your sessions, track progressive overload, and monitor your lifts.

## Live Demo
[https://flow-workout.onrender.com](https://flow-workout.onrender.com)

> Install as an app: on iPhone tap Share → Add to Home Screen. On Android tap ⋮ → Add to Home Screen.

## Features
- 📋 Create and manage workout sessions (Push Day, Pull Day, etc.)
- 🏋️ Exercise library with muscle group filtering
- 📝 Log sets with reps, weight, and set number
- 📱 PWA — installable on iPhone and Android
- 🌙 Apple-style dark UI

## Tech Stack
**Backend:** Java 17 + Spring Boot 4 + Spring Data JPA + PostgreSQL (Supabase)

**Frontend:** React PWA, mobile-first CSS

**Deployed on:** Render + Supabase
